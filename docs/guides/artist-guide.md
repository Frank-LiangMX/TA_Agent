# 美术使用指南

> 面向美术同学的快速上手教程，不讲原理，只讲操作。

---

## 一、快速启动

### 启动方式

找到 `F:\ta_agent\` 根目录，双击 `dev-web.bat`（浏览器模式）或 `dev-electron.bat`（桌面应用模式）。

等终端出现如下提示就说明启动完成：

```
后端已启动: http://localhost:8080
前端已启动: http://localhost:5175
```

然后打开浏览器，访问 **http://localhost:5175** 即可进入主界面。

> 关闭时双击 `stop-web.bat` 即可。

### 界面布局

```
┌──────────┬──────────────────────────────┬──────────┐
│ 左侧导航  │      中间主面板              │ 右侧详情  │
│          │                              │          │
│  对话     │  在这里跟 AI 对话            │ 查看资产  │
│  资产库   │  "分析 D:\我的资产"          │ 的详细信息 │
│  审核     │                              │          │
│  搜索     │                              │          │
│  设置     │                              │          │
└──────────┴──────────────────────────────┴──────────┘
```

左侧可以切换页面，中间是主要操作区，右侧看资产详情。

## 二、日常使用流程

### 场景 1：检查一批新资产

做完一批模型/贴图后，想检查有没有问题。

```
① 在对话输入框输入：
   帮我看一下 D:\我的资产\角色  这批文件有没有问题

② AI 会自动：
   ├── 扫描目录下所有文件
   ├── 解析 FBX 模型（面数、骨骼、UV）
   ├── 检查贴图（分辨率、格式）
   ├── 检查命名是否规范
   └── 给出分析报告

③ 查看结果：
   ✅ 通过的项目 → 没问题，直接下一步
   ⚠️ 警告的项目 → 看看哪里有问题
   ❌ 失败的项目 → 需要修改
```

### 场景 2：搜索以前做过的资产

想找一个以前做过的资产，但忘了文件名。

```
在左侧点击"搜索"，然后输入：

  我需要一个中世纪的铁剑，带磨损效果
  
AI 会从已入库的资产中找出匹配的结果，按匹配度排序。
```

### 场景 3：批量改名

有一批文件命名不规范，想统一改。

```
在对话框输入：

  帮我把 D:\我的资产\武器 目录下的所有文件按规范重命名
  
AI 会先列出建议的新名称，你确认后执行。
```

### 场景 4：导入到 UE5

检查通过的资产想要导入到 UE5。

```
① 先确保 UE5 已打开，并且在 Python Console 中执行：
   exec(open(r"F:\ta_agent\ue5_server\server.py").read())

② 在对话框输入：
   把刚才审核通过的那些资产导入到 UE5 的 /Game/Characters/ 目录

③ AI 会自动执行导入，完成后会告诉你结果。
```

## 三、常见问题

### Q：启动时提示"API Key 未配置"

联系 TA 帮你配置 `config.py` 中的 API Key。

### Q：分析结果不准确

AI 推断不准确是正常的。你可以：

```
① 在审核页面找到这个资产
② 点击"修改"，把正确的分类/材质/风格填上
③ 提交修改

AI 会记住这次纠正，下次同一个类型的资产就会更准。
```

### Q：UE5 导入失败

检查以下几点：

1. UE5 是否已打开
2. 是否在 Python Console 执行了启动命令
3. 目标路径是否正确（应该是 UE5 的 Content 子目录，如 `/Game/Characters/`）

如果还是不行，把错误提示截图发给 TA。

### Q：分析到一半卡住了

可能是文件太大或者 UE5 连接超时。在对话框输入"停一下"或直接刷新页面重新开始。

## 四、常用指令速查

| 你想做什么 | 对 AI 说 |
|-----------|---------|
| 检查一批文件 | `分析 D:\路径` |
| 只看模型面数 | `检查 D:\路径 下的模型面数` |
| 只看贴图 | `检查 D:\路径 下的贴图规格` |
| 检查命名 | `检查 D:\路径 下的文件名是否规范` |
| 搜索资产 | 在搜索页直接输入自然语言 |
| 审核资产 | 点左侧"审核"进入审核页面 |
| 改名 | `帮我把 xxx 改名为 xxx` |
| 批量改名 | `按规范重命名 D:\路径 下的所有文件` |
| 导入 UE5 | `把资产导入到 /Game/xxx/` |

---

## 五、提示

- **用自然语言跟 AI 说话就行**，不需要记命令格式
- **AI 推断错了就纠正它**，纠正一次下次就更准
- **先让 AI 检查再导入**，避免把有问题的资产弄进引擎
- 遇到问题问 AI 本身，或者找 TA 帮忙
